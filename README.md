# P2HACKS2025 アピールシート

## プロダクト名

biestar

## コンセプト

何かを始める(キラキラしている)人が先人の経験を学び、活かせる場所

## 対象ユーザ

初めたいことがあるけど相談できる人がいない…
何からやればいいかわからない…　　なんて人に利用していただきたい

こんなことやったことある！、やったけどこうすればよかったな…他の人にこんな思いしてほしくないな…
なんて人の経験談もほしい！

## 利用の流れ

プロダクトを起動したり、ランディングした時やその後の流れを書きます。
手順書のようなものではなくてもよいですが、読んだらプロダクトの使い方がわかるものがよいです。
必要に応じて、プロダクトを知るまでの流れも書いてよいでしょう。

## 推しポイント

知恵袋のように１個の質問に１個の回答ではなくつながりから聞きやすい状態を続けてサポートしてもらえる環境
一般的に良くないとされることの多い自分語りを他人の役に立てられるサイト
